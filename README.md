# tele-sync

## 概要
tele-syncは3D Gaussian Splatting (3DGS)と非対称MRを用いた遠隔コミュニケーションシステムです。現地のユーザーが実環境をキャプチャして、そのデータから3DGSによって、3Dシーンの構成を行います。そして、実環境を反映させたバーチャル環境でのコミュニケーションを可能にします。
